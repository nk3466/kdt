# 스마트 스피커 프로젝트

이 프로젝트는 Python을 사용하여 간단한 스마트 스피커를 구현합니다. 음성 인식(STT), 음성 합성(TTS), 음성 활동 감지(VAD) 기능을 포함합니다.

## 기능

- **음성 인식 (STT)**: 사용자의 음성을 텍스트로 변환
- **음성 합성 (TTS)**: 텍스트를 음성으로 변환
- **음성 활동 감지 (VAD)**: 음성이 있는 부분만 녹음

## 파일 설명

- `smart_speaker.py`: 메인 스마트 스피커 애플리케이션
- `stt.py`: 음성 인식 기능 (Google Speech Recognition 사용, 한국어 지원)
- `tts.py`: 음성 합성 기능 (pyttsx3 사용, 한국어 음성 출력)
- `vad.py`: 음성 활동 감지 기능 (WebRTC VAD 사용, 실시간 음성 감지 및 녹음)
- `translate.py`: 번역 기능 (미구현)
- `requirement.txt`: 필요한 패키지 목록

### stt.py 상세 설명
- Google Speech Recognition API를 사용하여 WAV 파일에서 텍스트를 추출합니다.
- 한국어('ko-KR')로 설정되어 있습니다.
- 입력 파일: 'vad_recorded.wav' (VAD에서 생성된 음성 파일)
- 오류 처리: 파일 존재 여부, 인식 실패, 서비스 접근 오류 등

### tts.py 상세 설명
- pyttsx3 라이브러리를 사용하여 텍스트를 음성으로 변환합니다.
- 음성 속도(180), 볼륨(0.8) 설정
- 한국어 음성을 우선적으로 선택 (사용 가능한 경우)
- 샘플 텍스트: "안녕하세요, 스마트 스피커입니다. 무엇을 도와드릴까요?"

### vad.py 상세 설명
- WebRTC VAD를 사용하여 실시간으로 음성을 감지합니다.
- 민감도 설정: 3 (가장 민감)
- 오디오 설정: 16kHz, 모노, 20ms 청크
- 녹음 시간: 10초
- 음성 감지 시 시작/종료 시간 출력
- 전체 녹음 파일('full_recorded.wav')과 음성 부분만('vad_recorded.wav') 저장
- Windows에서 자동 재생 기능 포함

## 설치 및 실행

1. 필요한 패키지 설치:
   ```
   pip install -r requirement.txt
   ```

2. 메인 애플리케이션 실행:
   ```
   python smart_speaker.py
   ```

## 사용법

1. VAD를 사용하여 음성을 녹음합니다.
2. 녹음된 음성을 STT로 텍스트로 변환합니다.
3. 변환된 텍스트를 처리합니다 (현재는 간단한 응답).
4. TTS를 사용하여 응답을 음성으로 출력합니다.

## 주의사항

- 인터넷 연결이 필요합니다 (Google STT 서비스 사용).
- 마이크가 연결되어 있어야 합니다.
- Windows 환경에서 테스트되었습니다.
